Frigate Notify Bridge

A Home Assistant integration that bridges Frigate NVR events to mobile push notifications for the Frigate Mobile app.

Features

  • Real-time push notifications for Frigate detection events
  • Multiple push providers: Firebase Cloud Messaging, ntfy, Pushover
  • QR code pairing with the Frigate Mobile app
  • Automated Firebase setup via Google Sign-In

Installation

Install via HACS or download from the releases page.
